# 🎴 TCG Collection App

**TCG Collection App** adalah aplikasi mobile berbasis **Flutter** untuk mengelola koleksi kartu **Trading Card Game (TCG)**, khususnya kartu **Pokémon**.  
Pengguna dapat menyimpan, melihat, dan mengatur koleksi kartu mereka secara digital, dengan dukungan autentikasi akun dan penyimpanan cloud melalui **Supabase**.

---

## 🌟 Fitur Utama

### 🔐 Autentikasi Pengguna
- Register, Login & Logout
- Auto-login jika session masih aktif
- Ganti password dari halaman profil

### 🃏 Manajemen Kartu
- Tambah kartu baru ke koleksi
- Edit informasi kartu kapan saja
- Upload gambar kartu
- Hapus kartu dari koleksi

### 📊 Tampilan Koleksi
- **List View**, **Grid View**, dan **Large View**
- Detail kartu: nama, series, rarity, harga, jumlah, ID

### 🔍 Pencarian & Filter
- Cari kartu berdasarkan nama
- Filter berdasarkan rarity

### 📈 Statistik Koleksi
- Total jumlah kartu
- Total nilai koleksi
- Informasi rarity yang sedang difilter

### 👤 Profil Pengguna
- Edit nama dan foto profil
- Dapat badge kolektor berdasarkan jumlah kartu

### 🌗 Tema Aplikasi
- Light Mode & Dark Mode
- Toggle dari halaman profil

---

## 🛠 Teknologi & Tools
- **Flutter & Dart** – Frontend & logic
- **Supabase** – Auth & Database
- **Provider** – State Management
- **VSCode / Android Studio** – IDE
